# BMP Modpacks Translations — StoneBlock 4

Локальные переводы для модпака **FTB StoneBlock 4**.  
Основной репозиторий: [BMProjects-Development/BMP-Modpacks-Translations](https://github.com/BMProjects-Development/BMP-Modpacks-Translations/tree/main)

## Использование

### Linux

```sh
git clone https://github.com/pixdns/BMP-Modpacks-Translations-StoneBlock4.git
cp -r BMP-Modpacks-Translations-StoneBlock4/* .
````

### Windows (PowerShell)

```powershell
git clone https://github.com/pixdns/BMP-Modpacks-Translations-StoneBlock4.git
Copy-Item "BMP-Modpacks-Translations-StoneBlock4\*" "." -Recurse -Force
```
